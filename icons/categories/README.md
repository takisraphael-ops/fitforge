# Category icons

Transparent PNGs, one per category key, used by the exercise-picker tiles.
Any missing file falls back to a built-in line icon automatically.

Expected files (FitForge category keys):
  arms.png  back.png  chest.png  core.png  shoulders.png  legs.png  cardio.png  all.png
  (full_body.png optional — falls back to a line icon)
